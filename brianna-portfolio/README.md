# Brianna Osborne Portfolio

A simple GitHub Pages portfolio website.

## How to publish with GitHub Pages

1. Create a new GitHub repository named `yourusername.github.io`.
2. Upload all files from this folder into the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. Your site should publish at `https://yourusername.github.io`.

## Edit before publishing

In `index.html`, replace:
- `yourusername` with your GitHub username
- `yourlinkedin` with your LinkedIn URL
- `your.email@example.com` with your preferred email
